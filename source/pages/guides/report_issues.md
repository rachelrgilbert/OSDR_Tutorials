# Report Issues to OSDR

## Visit the OSDR Contact Page

If you get an error message or identify any issues with OSDR resources (e.g. [Submission](https://www.nasa.gov/osdr-submission-portal/), [Visualization](https://visualization.osdr.nasa.gov/), or [Sample Request](https://visualization.osdr.nasa.gov/nbisc/home/) Portals, the [BioData API](https://visualization.osdr.nasa.gov/biodata/api/), or [Training Resources](https://www.nasa.gov/opportunities-and-training-resources/)) or datasets hosted on the [Data Repository](https://osdr.nasa.gov/bio/repo/), visit the [Contact OSDR](https://www.nasa.gov/contact-osdr/) page and complete the form, as instructed below, to report the issue(s) to the OSDR team.
  

## Complete the Contact OSDR Form

Complete the [Contact OSDR](https://www.nasa.gov/contact-osdr/) form (depicted below) as follows:

- **Name:** Input your first and last name.
- **Email:** Input your email address.
- **Subject:** 
  - If you are having issues with OSDR resources, include the name of the resource and a 1-5 word summary of the issue (e.g. "Data Visualization: OSD-422 not available", "RadLab: File download error", "NBISC: Sample request page down", "GL4U: Cannot enroll in course", etc.).
  - If you identify an issue with an OSD dataset, include the OSD number and a 1-5 word summary of the issue (e.g. "OSD-249: Broken publication link", "OSD-576: Typos in assay table", "OSD-771: Request for GeneLab processed data", etc.).
- **Message:** Provide a short description of the issue. 
  - If reporting an issue with the [Submission](https://www.nasa.gov/osdr-submission-portal/) portal, please include your NASA guest account username, and the OSD number that was assigned to you (if available).
  - If reporting an issue with one of the [Visualization](https://visualization.osdr.nasa.gov/) portals, please include the specific request that was made that led to the issue and any relevant links, OSD numbers, instruments, and/or tissues included in the request.
  - If reporting an issue with an OSD dataset on the [Data Repository](https://osdr.nasa.gov/bio/repo/), please include the OSD number and the section(s) of the study page where the issue(s) was found.
  - If reporting an issue with an on-demand [Training Resource](https://www.nasa.gov/opportunities-and-training-resources/), please include the name of the course and your Canvas username.
  - If reporting an issue with the [BioData API](https://visualization.osdr.nasa.gov/biodata/api/), include a description of what you are trying to do, the API call attempted, and the platform used (e.g. Chrome browser, local Jupyter Notebook, etc.).
